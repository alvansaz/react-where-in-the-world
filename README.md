# react-where-in-the-world

As a practice, I've created a React app for 'REST Countries' API.

## Tools I used

* Vite 3.2.3
* React 18.2.0
* React Router 6.4.3
* Axios 1.1.3
* Tailwind CSS 3.2.4

## How to run in development mode

```bash
  yarn
  yarn dev
```
or

```bash
  npm install
  npm run dev
```
